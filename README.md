# WebCore
Main PerceptionAI Website

### Clone project use follwing command
```
git clone https://github.com/PerceptionAI/WebCore.git
```
change the branch to master

### Run the server
```
docker-compose build
docker-compose up -d
```

### visit this page 
```
127.0.0.1:8008/home/
```

### close the server
```
docker-compose down
```
